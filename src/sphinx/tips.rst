Generating Code Documentation With Pycco
----------------------------------------

.. code-block:: bash

    $ pip install pycco
    $ cd to/root/of/project
    $ pycco **/*.py   -p -i

https://realpython.com/blog/python/generating-code-documentation-with-pycco/
